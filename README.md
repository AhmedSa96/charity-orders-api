## Description

Charity Orders API is an open source api made using

- docker
- docker-compose
- multi-stage enviroments on docker
- unit testing
- postgres
- openApi/Swagger
- web socket
- rate limit security

maybe we could add 

- js report
- payment gateways


## Run as docker

```bash
# run dev
$ yarn docker-compose:dev
```

Or Localy

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```
